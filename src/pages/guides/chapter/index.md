# Гайды по Главам

Добро пожаловать в раздел гайдов, посвященных механике глав (сезонов) в игре Sunflower Land!

## О чем этот раздел

В этом разделе собраны все материалы, связанные с сезонной механикой игры. 
Каждый сезон (глава) приносит новые возможности, бонусы и условия для получения наград.
