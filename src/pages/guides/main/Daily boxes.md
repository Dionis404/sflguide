# Награды за вход

Награды за вход - это система, которая позволяет пользователям получать вознаграждения за вход в игру каждый день. 

## Первые 7 дней
Первые 7 дней для **новых игроков** — отдельный набор:  
**День 1:** Basic Farming Pack- набор семян/фермерских ресурсов  
**День 2:** Builder basics - ресурсы  
**День 3:** Basic Love Box - базовый Love бокс   
**День 4:** Tool tune up - инструменты  
**День 5:** Базовые монеты  
**День 6:** Fishing - предметы для рыбалки  
**День 7:** First week finale — Weekly Mega Box + Gem

## Остальные дни 
- **Coin Stash** - монеты (пример найденый в коде: Math.floor(500 * (level / 25)) — масштабируются с уровнем)
- **Growth Feast** - зелье опыта (количество масштабируется с уровнем) На 90 уровне зелье опыта даёт примерно 10% от опыта, который нужен до следующего уровня. Чем выше уровень персонажа, тем меньший процент опыта даёт зелье. 
- **Growth Boost** - Power hour (50% скорости роста, +0.2 к сбору урожай на 1 час ) 
- **Tool Cache** - инструменты (Axe, Pickaxe, Iron Pickaxe) — количества масштабируются с уровнем
- **Love Box** - Basic / Bronze / Silver в зависимости от уровня игрока 
- **Angler Pack** - рыболовные предметы: Rod, Earthworm, Grub (количества масштабируются)
